# Inventory Management System (Flask Web App)

A **modern web-based Inventory Management System** built with **Flask**, **SQLite**, and **Pandas**, featuring a clean web interface, user authentication, product management, search, low stock alerts, and CSV export.  

This project is ideal for **placement projects**, showcasing **Python, Flask, SQL, and web development skills**.

---

## **Features**

- User **Login & Logout** (Admin & other users)
- **Dashboard** with analytics:
  - Total Products
  - Low Stock Products
  - Inventory Value
- **Add/Delete Products**
- **Search products** by name or category
- **Low stock alerts**
- **Export inventory** to CSV
- Fully **responsive and styled interface** with HTML, CSS, JS

---

## **Tech Stack**

- **Backend:** Python, Flask
- **Database:** SQLite (relational database)
- **Data Handling:** Pandas
- **Frontend:** HTML, CSS, JavaScript
- **Version Control:** Git, GitHub
